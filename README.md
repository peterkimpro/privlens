# Privlens

**Private on-device AI document analyzer for iOS.**

Scan any document. Extract key insights. Everything stays on your device.

## What is Privlens?

Privlens uses Apple Foundation Models to analyze documents entirely on-device — your data never leaves your iPhone. No cloud APIs, no subscriptions to external AI services, no privacy trade-offs.

### Key Features

- **Document Scanning** — Camera-based scanning with automatic edge detection and perspective correction
- **AI-Powered Analysis** — Extract key clauses, hidden fees, important dates, and actionable insights
- **100% On-Device** — All AI processing runs locally via Apple Foundation Models (~3B parameter LLM, iOS 26+)
- **Privacy First** — Documents are never uploaded, transmitted, or stored in the cloud
- **Multi-Document Types** — Leases, contracts, medical bills, insurance policies, tax forms, and more

### Tech Stack

- **SwiftUI** — Modern declarative UI framework
- **Apple Foundation Models** — On-device LLM for document understanding
- **Vision Framework** — OCR and document scanning
- **VisionKit** — Camera-based document capture

## Development

Requires:
